
# OSeMOSYS Lao PDR Whole Energy Systems model (WESM)

his repository contains the OSeMOSYS Lao PDR Whole Energy System Model (WESM), developed within the Climate Compatible Growth (CCG) Programme under the Southern Partners' Fund (SPF) project PBFL.  The model represents the entire Laotian energy system, including the commercial, industrial (includes agriculture), residential and transport sectors. The model was developed using Modelling User Interface for OSeMOSYS [MUIO](https://github.com/OSeMOSYS/MUIO?tab=readme-ov-file) version 5.4. There are four model files that include a baseline and three scenarios that were developed to produce policy briefs during the OSeMOSYS Whole Energy Systems Modelling workshop in Lao PDR (March 2026). The model file and a short documentation of the data and model structure is also provided.
